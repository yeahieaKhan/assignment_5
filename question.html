Question-1: What are the different ways to select an element in the DOM?

DOM stands for Document Object Model. It is used to design web pages. It represents the page so that programs can change the document structure, style, and content. There are five ways to select elements in the DOM:

  1. Using getElementById.
  2. Using getElementsByClassName.
  3. Using getElementsByTagName.
  4. Using querySelector.
  5. Using querySelectorAll.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

textContents is all text contained within an element and all its child elements that are for formatting purposes only. innerText Returns all text contained within an element and all its child elements. innerHtml Returns all text contained within an element, including html tags.

Question-3: What is event delegation in the DOM?

Event Delegation is a useful technique in JavaScript, where basically an event listener is added to a parent element, so that it can handle the events of its child elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is a mechanism where when an event is triggered on a child element, it propagates to its parent and higher elements. This allows parent elements to respond to child element events.

Question-5: How do you create, add, and remove elements using JavaScript?

create element
const elements = document.getElementById("demo")
const div = document.createElement(div);
elements.appendChild(div);

Delete element
const elements = document.getElementById("demo")
